    Click here to open the project


  • Guess the Number Game


    • This is a simple yet interesting terminal based game written in python. In this game, the computer randomly generates a number in the range 1 to 100(both 1 and 100 are included) and you are supposed to find that particular number in the least possible number of turns using the hints given by the computer after every unsuccessful guess. This game requires you to have a pen and a paper and can be played by any person who knows basic mathematics.

    This game is highly benificial for primary/upper-primary school students, where they can learn and apply basic mathematics and enjoy the process of learning
